My Projects

Block Music

Block Music is an interactive experience that allows users to create music by physically arranging puzzle pieces that represent various musical pitches and note values. Designed for learners of all ages, the project helps teach the fundamentals of music creation in a hands-on way. I developed backend logic using Python, OpenCV, and Tesseract OCR to interpret and convert the puzzle arrangements into sound. The project was successfully demoed by over 100 elementary and middle school students in Evanston, enhancing their understanding of music and coding through play.

Secure To-Do List

This full-stack to-do list application was built using Next.js with a focus on security and scalability. The project integrates Docker to ensure consistent environment configurations and employs advanced security protocols, including cryptographic hashing and token-driven authentication, to protect user data. The application was developed using domain-driven design principles, with seamless backend and frontend integration. Manual testing was implemented for product security, along with Unix permission configurations to prevent unauthorized access and maintain system integrity.

Wildcat Track @ NU

This project focuses on utilizing OpenStreetMap (OSM) data to create a navigation system specifically tailored for Northwestern University's Evanston campus. The system reads and processes map data from XML documents, making it adaptable for other universities or communities. This data is used to build a graph representation of the campus, including nodes and footways, which are then leveraged to calculate optimal routes. Technologies used include C++, data structures (maps, vectors), XML parsing, and algorithms (graph traversal, shortest path).

Virtual Masks

This project allows users to create and customize virtual masks with dynamic features using p5.js and Vue.js. The virtual masks use linear interpolation to provide smooth and responsive animations, allowing for features like adjustable particle effects, spikes, and facial overlays that react to the user’s movements. These masks are designed to be both visually engaging and highly customizable, offering a range of adjustable parameters such as mask speed, particle lifespan, and more. Leveraging p5’s drawing capabilities and Vue’s reactive components, the masks can be tailored to suit each user’s personal aesthetic, creating an experience that is both interactive and unique.
